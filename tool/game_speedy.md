## 网游加速的原因分析
```
在国内，网游加速的原因分为两类。

第一类是由于不同网络运营商间的壁垒造成的。

第二类是由于游戏服务器在境外，使得玩家需要跨境进行数据交互。
```

## 网游加速可以采用两种方式来实现。
```
一种是VPN。

另一种是代理服务器。
```
